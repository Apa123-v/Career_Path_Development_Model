# AI Career Roadmap Generator

An AI Career Roadmap Generator built using React, Tailwind CSS, and Lucide Icons that dynamically displays personalized milestones and interactive practice mock popups.

---

## ⚡ Quick Start (Run in Under 2 Minutes)

Follow these simple steps to get the project running locally on your machine right after cloning.

### 1. Clone the Repository
```bash
git clone [https://github.com/Apa123-v/Career_Path_Development_Model.git](https://github.com/Apa123-v/Career_Path_Development_Model.git)
cd Career_Path_Development_Model
```
### 2. Set up & Run Backend (FastAPI)
```
# Navigate to backend directory
cd backend

# Install dependencies
pip install -r requirements.txt

# Start the server
uvicorn main:app --reload
(Backend will be live at: http://127.0.0.1:8000)
```
### 3. Setup & Run Frontend (React + Vite)
```
# Navigate to frontend directory
cd frontend

# Install dependencies
npm install

# Start the development server
npm run dev

(Frontend will be live at: http://localhost:5173)
```
🛠️ Tech Stack Used
*Frontend: React.js, Vite, Tailwind CSS
*Icons: Lucide React
*Backend: Python, FastAPI, Uvicorn
